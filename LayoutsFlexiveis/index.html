Layouts Flexíveis (Flexbox)

O que é Flexbox?

Flexbox é um recurso do CSS utilizado para organizar elementos de maneira flexível dentro de um container.

Ele facilita o alinhamento, espaçamento e distribuição dos elementos em linhas ou colunas, sendo muito utilizado no desenvolvimento de interfaces modernas e responsivas.

Como funciona?

Para utilizar o Flexbox, é necessário aplicar display: flex em um elemento container.

A partir disso, os elementos internos podem ser alinhados, centralizados e organizados de diversas formas utilizando propriedades como:

  • justify-content
  • align-items
  • flex-direction
  • gap

Exemplo de código


.container {

    display: flex;

    justify-content: center;

    gap: 20px;
}
