Media Queries

O que são Media Queries?

As Media Queries são recursos do CSS utilizados para tornar páginas web responsivas, permitindo que o layout se adapte a diferentes tamanhos de tela, como celulares, tablets e computadores.

Com elas, é possível alterar estilos dependendo da largura, altura, resolução ou orientação da tela do dispositivo. Esse recurso é muito utilizado no desenvolvimento web moderno para melhorar a experiência do usuário.

Como funcionam?

As Media Queries funcionam verificando características do dispositivo. Quando determinada condição é atendida, o CSS aplica estilos diferentes.

Um exemplo comum é alterar o layout quando a largura da tela fica menor que 800 pixels, adaptando o conteúdo para celulares.

Exemplo de Media Query

Exemplo de código: 


@media (max-width: 800px) {

    body {

        background-color: red;

    }
}

Nesse exemplo, quando a tela possuir largura menor que 800px, a cor de fundo da página será alterada.

Exemplo Prático

Redimensione a janela do navegador para visualizar as alterações de estilo acontecendo em tempo real através das Media Queries. Caso esteja utilizando um computador, clique com o botão direito na página, selecione “Inspecionar” e depois clique no ícone de dispositivos móveis para simular a visualização em celulares e tablets.
