CSS Grid Layout Profissional

O que é CSS Grid Layout?

O CSS Grid Layout é um recurso do CSS criado para facilitar a construção de layouts modernos e organizados em páginas web.

Ele permite dividir a página em linhas e colunas, tornando muito mais fácil posicionar elementos de forma precisa e responsiva.

O Grid é bastante utilizado em sistemas administrativos, dashboards, galerias de imagens, páginas institucionais e diversos outros tipos de interfaces modernas.

Como funciona?

Para utilizar o Grid Layout, primeiro definimos um container com display: grid.

Depois disso, podemos criar colunas e linhas utilizando propriedades como:

  • grid-template-columns
  • grid-template-rows
  • gap
  • justify-content
  • align-items

Isso permite criar estruturas organizadas sem precisar utilizar vários posicionamentos complexos.

Exemplo de Código


.container {

    display: grid;

    grid-template-columns: 1fr 1fr 1fr;

    gap: 20px;
}

.item {

    background-color: steelblue;

    color: white;

    padding: 20px;
}

Nesse exemplo, o container foi dividido em 3 colunas iguais utilizando a propriedade grid-template-columns.
